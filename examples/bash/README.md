# Bash

Sets the default shell to the _Bourne Again SHell_ (`bash`). Run this example with:

```bash
nix run ./examples/bash
```

> [!WARNING]
> Do not run this command from _inside_ of `rv`, as this will just inherit the properties of the 
> _current_ editor you are in. No [editorception]!
