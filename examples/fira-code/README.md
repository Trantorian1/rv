# Fira-code

Sets the default graphical font to [fira-code], a popular programming font. Run this example with:

```bash
nix run ./examples/fira-code
```

> [!WARNING]
> Do not run this command from _inside_ of `rv`, as this will just inherit the properties of the 
> _current_ editor you are in. No [editorception]!

[fira-code]: https://github.com/tonsky/FiraCode
